# 🍽️ Los Gatos Cocineros

Recetario web pensado para personas recientemente diagnosticadas con celiaquía, ofreciendo opciones de comidas seguras, simples y accesibles.

## 📌 Descripción

**Los Gatos Cocineros** es una aplicación web que busca acompañar a personas que recibieron recientemente el diagnóstico de celiaquía, ayudándolas a adaptarse a su nueva alimentación mediante recetas libres de gluten.

El proyecto se centra en brindar una experiencia clara, amigable y educativa, reduciendo la incertidumbre inicial que suele acompañar este cambio de hábitos.

---

## 🎯 Objetivo

Facilitar el acceso a recetas **sin gluten**, promoviendo una alimentación segura y accesible para personas con celiaquía.

---

## 🚀 Características

- 🥗 Recetas aptas para celíacos
- 🔍 Exploración sencilla de contenido
- 📱 Diseño responsive
- ⭐ Interfaz clara para usuarios principiantes
- 🧾 Presentación organizada de ingredientes y preparación

---

## 🛠️ Tecnologías utilizadas

- React
- Node.js
- Tailwind
- MongoDB
